== Description ==

[LIVE DEMO](https://demos.infosofttech.com/subscriptiondemo/wp-login.php) | [PRO VERSION LINK](http://plugins.infosofttech.com/products/woocommerce-subscription-report/)

[plugins.infosofttech.com](http://plugins.infosofttech.com/) presents WooCommerce Subscription Report Plugin, this works with [WooCommerce Subscriptions](https://woocommerce.com/products/woocommerce-subscriptions/) Plugin ver 2.0.8 onwards.

**Features**: 
Dashboard shows summary like Total Subscribed User Count, Active Subscriber Count, Active Subscription Count, Paid signup Count, Top n Subscription Item wise summary, WooCommerce Order Status and Subscription Status wise summaries.

Daily summary report with order information, Report can be filtered by Date.


**Additional Features in Pro Version**:
Various counts like Total Free Signup, Total Paid Sunscriber, Total Cancelled Subscriptions, New Free Trial Signup, Total Revenue on Dashboard, graphical presentation of Revenue, Free Trials, Paid Subscribers and Cancellations by Days/Week/Month, different summaries like Free/Trial Subscription Due, Top n Subscription Country, Month wise Renewal Due, Top n Coupons, Subscription Due, Subscription Expired, Renewals Due, Renewals Overdue, Product/Month Crosstab report.

Detailed Subscription report having Date, Customer, Email, Country, Subscription Status and Order Status filters. All the data can be exported to Excel, CSV.

Subscription summary like count and amount of New Subscriptions, New Renewals, New Cancellations same for Products as well.


[Click here](https://plugins.infosofttech.com/products/woocommerce-subscription-report/) to know more about Pro Version

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins > Add New > Search` and search for 'WC Subscriptions Report Lite').
* Alternatively, you can [download](http://downloads.wordpress.org/plugin/ic-woocommerce-subscriptions-lites.zip "Download WooCommerce Colored Order Notes") the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Access the settings page from WordPress backend under WC Subscriptions Report Lite.
